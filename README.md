# 👩‍💻 Bianca Lau – Software Engineering Projects

This repository contains a collection of academic projects developed throughout my Bachelor of Computer Science (Software Engineering) studies.

The projects cover multiple domains including system development, database design, simulation, operating systems, and low-level programming.

---

## 📌 Project Categories

### 🔹 Core Systems & Application Development
- **Parking Lot Management System (Java, Swing, SQLite)**  
  A GUI-based system supporting real-time parking operations, pricing logic, and database integration.

- **Seminar Management System (Java, Swing)**  
  A role-based system supporting scheduling, submission, and evaluation workflows.

- **EduTrack Attendance & Analytics System (Java, Tomcat, MySQL)**  
  A web-based system with QR-based attendance tracking, analytics, and reporting features.

- **Monopoly OS (C, Linux)**  
  A multi-client system using sockets, threading, and IPC with process synchronization.

---

### 🔹 Algorithms, Simulation & Problem Solving
- **Robot War Simulator (C++)**  
  A grid-based robot simulation with multiple robot behaviors using object-oriented design.

- **Petrol Station Simulation System (MATLAB / FreeMat)**  
  A queue-based simulation with performance analysis under varying traffic conditions.

- **Programming Fundamentals in C++**  
  A collection of basic programs focusing on loops, arrays, and structured problem solving.

---

### 🔹 Database Systems & Data Modeling
- **Cinema Booking Database System (SQL)**  
  A relational database with ERD design, normalization, and integrity constraints.

---

### 🔹 Low-Level Programming & Digital Systems
- **Date Validation System (ARM Assembly)**  
  A low-level program for validating BCD-encoded dates using bit manipulation.

- **Digital Logic Circuit Design (CircuitVerse)**  
  Logic circuit design using flip-flops, K-map simplification, and state transitions.

---
